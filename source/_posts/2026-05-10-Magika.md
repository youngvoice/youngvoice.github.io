# content

Magika employs a deep learning model that can execute on a single CPU with just 1MB of memory to store the model's weights. The model is mainly used to do the task of content-type detection which entails identifying the data encoded in an arbitrary byte sequence. This task is mainly involved in reverse engineering environments, and a variety of security applications.
